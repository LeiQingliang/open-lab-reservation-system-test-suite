## 1. 系统概述

### 1.1 系统基本信息

| 属性 | 值 |
|------|-----|
| 系统名称 | 开放式实验室网上预约管理系统 |
| 英文名称 | Open Lab Reservation System |
| 基础框架 | 若依(RuoYi) v3.8.9 |
| 架构模式 | 前后端分离 (SPA + RESTful API) |
| 开发语言 | Java (后端) + JavaScript/Vue3 (前端) |
| 作者 | santh |
| 开发日期 | 2025-04 |

### 1.2 系统定位

本系统是一个面向高校的**实验室资源在线预约管理平台**，实现实验室信息的发布、浏览、预约申请、审核管理等功能。系统基于若依开源框架进行二次开发，保留了框架原有的用户管理、角色权限、日志监控等基础能力，并在此基础上新增了实验室管理、排期发布、预约申请、预约审核等核心业务模块。

### 1.3 核心业务能力

```
┌──────────────────────────────────────────────────────────────┐
│                    核心业务能力矩阵                            │
├──────────────┬───────────────────┬────────────────────────────┤
│   能力域      │   功能描述          │   涉及角色                  │
├──────────────┼───────────────────┼────────────────────────────┤
│ 实验室管理    │ 实验室信息CRUD       │ 超级管理员(admin)           │
│ 排期发布      │ 发布可预约时段        │ 超级管理员(admin)           │
│ 预约申请      │ 学生/教师提交预约     │ 普通用户、教师              │
│ 预约审核      │ 审核通过/拒绝预约     │ 教师                       │
```

## 7. 安全架构分析

### 7.1 认证流程

```
请求到达 → JwtAuthenticationTokenFilter
    │
    ├─ Token不存在 → 放行（由后续Security处理）
    │
    └─ Token存在 → 解析JWT
            │
            ├─ 解析失败 → 返回401
            │
            └─ 解析成功 → 获取userId
                    │
                    ├─ Redis中Token不存在 → 返回401
                    │
                    └─ 从Redis获取用户信息 → 构建Authentication
                            │
                            └─ 存入SecurityContextHolder
```

### 7.2 鉴权机制

- **方法级鉴权：** 使用 `@PreAuthorize("@ss.hasPermi('xxx:xxx:xxx')")`
- **权限字符串格式：** `模块:实体:操作`（如 `reservation:reservation:list`）
- **角色体系：** RBAC（用户→角色→菜单/权限）
- **匿名访问：** 通过 `@Anonymous` 注解标记 + `PermitAllUrlProperties` 配置

### 7.3 安全配置清单

| 配置项 | 值 | 说明 |
|--------|-----|------|
| CSRF | 禁用 | 前后端分离无需CSRF |
| Session | STATELESS | 基于JWT，无状态 |
| 密码编码 | BCryptPasswordEncoder | 强散列哈希 |
| Token有效期 | 30分钟 | `token.expireTime: 30` |
| Token存储 | Redis | 支持主动失效 |
| 密码最大错误次数 | 5次 | 锁定10分钟 |
| XSS过滤 | 启用 | 排除 `/system/notice` |
| 验证码 | 数学计算类型 | 登录时使用 |

### 7.4 安全漏洞分析

| 风险点 | 严重程度 | 说明 | 代码位置 |
|--------|----------|------|----------|
| Token未加密存储 | 中 | Token在Redis中明文存储 | RedisConfig |
| 无Rate Limiting | 中 | 登录接口无频率限制（虽有@RateLimiter但未实际应用于登录） | SecurityConfig |
| SQL注入 | 低 | MyBatis使用#{}预编译，注入风险低 | Mapper XML |
| XSS | 低 | 有XSS过滤器，但部分路径排除 | XssFilter |
| 敏感信息日志 | 中 | 操作日志记录完整请求参数，可能含密码 | LogAspect |
